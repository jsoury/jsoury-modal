## A small but functional npm package

### Install

Install using `npm install jsoury-modal`

### Usage

In a React app, use the Modal components:  
`import { Modal } from 'jsoury-modal'`

#### Button props

| Name     | Description |
| -------- | ----------- |
| show     | Boolean     |
| onClose  | Boolean     |
| title    | string      |
| children | string      |
